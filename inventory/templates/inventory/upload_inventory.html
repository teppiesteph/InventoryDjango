Upload Inventory

Upload a .txt file to bulk add products to your inventory.

File Format Instructions

Ensure your file is formatted as follows:

  • Each line represents one product.
  • Each line should contain the following details, separated by commas:
    • Product Name
    • Product ID (unique)
    • Description
    • Amount (integer)
    • Location
  • Example of a valid file:
Laptop,1001,High-performance laptop,10,Warehouse A
Mouse,1002,Wireless mouse,50,Warehouse B
Keyboard,1003,Mechanical keyboard,20,Warehouse C

Note: The file must be in .txt format. Other file types are not supported.
